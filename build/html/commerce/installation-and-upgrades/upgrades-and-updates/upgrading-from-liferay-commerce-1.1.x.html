Upgrading from Liferay Commerce 1.1.x¶

This article documents how to upgrade Liferay Commerce to the latest maintenance version. Store administrators should consider regularly updating to the latest available maintenance version to receive bug fixes and new features.

Roadmap¶

  1. Download and Deploy.

  2. Clear Stale Data and Verify the Upgrade Process.

  3. Execute Post-Upgrade Reindex.

Download and Deploy¶

  1. Download the latest Liferay Commerce LPKG from Help Center.

    The latest open source LPKG may be obtained from the Liferay Commerce Community Download Page.

  2. Deploy the LPKG to the ${liferay.home}/deploy folder.

    To learn more about deploying applications to Liferay DXP, see Liferay Home.

  3. Verify that that the following message appears in the application server console:

Execute Post-Upgrade Reindex¶

After upgrading from Liferay Commerce 1.1.x to the latest version, it is necessary to execute a full search reindex because of product changes between major versions.

To execute a search reindex:

  1. Navigate to the Control Panel → Configuration → Search.

  2. Click Execute next to Reindex all search indexes.

  3. Wait for the reindex to finish.

  4. Navigate to the Control Panel → Commerce → Products.

  5. Verify all the products are displayed again.

Once reindexing is complete, the new Liferay Commerce instance is ready for use.
