Authorize.net¶

This article details how to enable Authorize.Net as a payment method. Authorize.Net is a subsidiary of Visa and a provider of payment gateway services that manages the submission of billions of transactions to processing networks on behalf of merchant customers.

Before configuring your store to use Authorize.Net, it is necessary to generate an API Login ID and a Transaction Key.

  1. Navigate to Site Administration → Commerce → Settings → Payment Methods.

  2. Click Authorize.Net.

  3. Click Configuration.

  4. Enter the following:

    • API Login ID

    • Transaction Key

  5. Select Production if it is a live site or Sandbox if it is a test environment.

  6. Check the following options to enable to display the following:

    • Show Bank Account

    • Show Credit Card

    • Show Store Name

  7. Check the following options to enable to display the following:

    • Require CAPTCHA

    • Require Card Code Verification

  8. Click Save.

  9. Click the 3-dot icon next to Authorize.Net then Activate.

Once finished, Authorize.Net is now enabled.
