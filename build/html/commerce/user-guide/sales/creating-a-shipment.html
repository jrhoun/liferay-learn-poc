Creating a Shipment¶

This article describes how to create a shipment once an order has been received.

Creating a New Shipment¶

  1. Navigate to the Control Panel → Commerce → Shipments.

  2. Click the (+) button to add a new shipment.

  3. Select an order from the dropdown menu (for example, 41241).

  4. Click Save.

  5. Enter the quantity from the corresponding warehouse.

  6. Click Save.

  7. Verify the shipping address.

  8. Click Save.

The new shipment has been created.

Once a shipment has been created, there are several further actions an order manager may do with a shipment.

Verify the Items¶

  1. Click the Items tab to verify that the items are correct.

Update the Shipping Status¶

  1. Click the Details tab.

  2. Enter the name of the carrier in the Carrier field (for example: USPS, FedEx, UPS)

  3. Enter the Tracking Number.

  4. Select a status from the Status dropdown menu (for example: Ready to be Shipped).

  5. Click Save.

The order manager can update the status of the shipment as needed, until the order is completed.
